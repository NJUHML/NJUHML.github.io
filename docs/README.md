## NJUHML研究组简介：
水文气象研究组（Hydrometeorology Research Group）隶属于[南京大学大气科学学院][NJUatmospheric]，现位于南京大学仙林校区大气科学学院院楼。负责人是[袁慧玲][袁慧玲个人主页]教授。本研究组主要从事水文气象数值模拟、AI气象模型与深度学习算法等方向的研究工作。研究课题基于研究组总体发展方向与成员研究兴趣设立，具体研究通过每周组会及日常与导师的深入讨论共同推进。

## NJU-Earth模型简介：
[NJU-Earth](https://github.com/NJUHML/NJU-Earth)（南京大学AI全球大气预报系统）由团队联合阿里巴巴达摩院，基于 Baguan 架构共同研发的全球0.25°高分辨率AI气象预报模型。在此基础上，团队进一步构建了NJU-Earth系列化AI预报体系，形成BaguanHR、RainCast、RegionalCast、BaguanCyclone、CBC-Diffusion概率降水模型、NJU-Mars等专业子模型，为气象科研与防灾减灾等提供智能化技术支撑。
